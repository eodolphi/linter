Linter Command Line Client
===============================

version number: 0.0.1
author: Ernst Odolphi

Overview
--------

A python package that can be installed with pip.

Installation / Usage
--------------------

To install use pip:

    $ pip install linter

Or clone the repo:

    $ git clone https://github.com/eodolphi/linter-client.git
    $ python setup.py install
